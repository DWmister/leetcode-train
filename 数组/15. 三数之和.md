 [15. 三数之和](https://leetcode-cn.com/problems/3sum/)

给你一个包含 n 个整数的数组`nums`，判断`nums`中是否存在三个元素 a，b，c ，使得 a + b + c = 0 ？请你找出所有满足条件且不重复的三元组。

注意：答案中不可以包含重复的三元组。

**示例 1:**

```
给定数组 nums = [-1, 0, 1, 2, -1, -4]，

满足要求的三元组集合为：
[
  [-1, 0, 1],
  [-1, -1, 2]
]
```

**解题:**

```js
/**
 * @param {number[]} nums
 * @return {number[][]}
 */
var threeSum = function(nums) {
    /**
     * 方法：双指针
     * 一个指向当前循环元素的下一个元素，一个指向数组最后一个元素
     * 1、先对数组进行排序，如果第一个元素就大于0，则不可能存在三个元素相加为0
     * 2、如果三个元素相加大于0，则right--
     * 3、如果三个元素相加小于0，则left++
     */
    let len = nums.length
    let result = []
    if (len < 3) {
        return result
    }
    nums.sort((a, b) => a - b)
    if (nums[0] > 0) {
        return result
    }
    let left = 0, right = len - 1
    for (let n = 0; n < len - 1; n ++) {
        if (nums[n] === nums[n - 1]) {
            continue
        }
        left = n + 1
        right = len - 1
        while (left < right) {
            if (nums[n] + nums[left] + nums[right] === 0) {
                result.push([nums[n], nums[left], nums[right]])
                while (left < right && nums[left] === nums[left + 1]) {
                    left ++
                }
                while (left < right && nums[right] === nums[right - 1]) {
                    right --
                }
                right --
                left ++
            } else if (nums[n] + nums[left] + nums[right] > 0) {
                right --
            } else {
                left ++
            }
        }
    }
    return result

};
```
